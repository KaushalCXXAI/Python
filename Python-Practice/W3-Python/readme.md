# W3Schools Python Extras

This folder is a focused space for learning and practicing Python concepts that were not fully covered in the CS50 Python course — especially built-in functions, methods, and features essential for effective problem solving.

## Why this folder exists

While CS50 gave me a solid foundation in Python, I noticed that some utility functions and built-in tools (like advanced list methods, dictionary tricks, file I/O variations, string formatting, etc.) weren’t explored in depth.

To fill those gaps, I'm using **W3Schools' Python tutorial** as a reference and coding along for:

- Mastering **important in-built functions**
- Reinforcing **language features** I might’ve skipped or forgotten
- Prepping myself for better performance in **problem-solving platforms** like LeetCode, Codeforces, etc.

## Structure

Each file inside this folder corresponds to a specific topic or section from W3Schools. I’m not trying to complete the entire W3 site but only the stuff that’s relevant and helps me code smarter and faster.

## Note

This is **not a full-blown course** or project folder. It’s more like a **toolkit or supplement** to CS50 — a place to drill down into things like:
- `map()`, `filter()`, `zip()`, `enumerate()`
- List comprehensions and slicing
- Dictionary and set operations
- String methods and formatting
- Lambda functions and more

Basically, stuff that I’ll probably need *again and again* in real-world coding and competitive programming.
